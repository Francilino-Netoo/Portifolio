Sobre mim

Sou um Programador Full Stack, tenho experiência e conhecimento criando apps usando React Native, JavaScript e Java. Além disso, sou especialista em automação web e web scraping usando Python, muitas vezes, uso a junção dessas ferramentas para fazer automações complexas em sites. Hoje, a maioria dos meus clientes são pessoas que gostam de apostar em cassinos online; muitos pedem para fazer uma automação específica para cada jogo.
